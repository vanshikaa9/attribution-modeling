# Digital Marketing Attribution Modeling

This project implements multiple attribution models to analyze how different marketing channels contribute to conversions.

## Models Used
- First-Touch
- Last-Touch
- Linear
- Time-Decay
- ML-Based (Gradient Boosting)

## Features
- Synthetic customer journey generation
- Channel-wise revenue attribution
- Model comparison
- Visualization outputs

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## How to Run
```bash
pip install -r requirements.txt
python attribution_modeling_project.py
```
